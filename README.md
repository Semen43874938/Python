Описание 
Приложение для конвертации валют с графическим интерфейсом, использованием внешнего API и
сохранением истории операций. 
Получение API‐ключа 
1.Перейдите на сайт exchangerate-api.com.
2.Зарегистрируйтесь и получите бесплатный API‐ключ.
3.Замените YOUR_API_KEY в коде на ваш ключ.
